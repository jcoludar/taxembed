# Archive

Historical development notes, debug scripts, and legacy code preserved for
reference. Nothing in this directory is imported by active training or CLI code.

## Contents

- **Markdown files** &mdash; Intermediate development docs from the v1&ndash;v10a
  architecture evolution (iteration notes, debug sessions, scaling analyses).
- **`debug_scripts/`** &mdash; One-off debugging and analysis scripts from
  development sessions.
- **`legacy_facebook/`** &mdash; Original Facebook Research Poincar&eacute;
  embeddings code (`hype/` library and `embed.py` wrapper).
- **`legacy_scripts/`** &mdash; Scripts superseded by the `taxembed` CLI or
  replaced by newer implementations.
